HTML LEARNINGS

All HTML tags go inside the "html" element
"title" tag content is displayed on the browser's title bar
"div" tag divides content across sections
"b" tag makes text bold

CSS

CSS adds style to the HTML document
"link" tag can be used to set the CSS file to be used
Child elements inherit styles from its parent
Different metrics like px, %, pt can be used to specify length
